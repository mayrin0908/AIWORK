================================================================
Claude Code コマンド一覧
================================================================

■ CLIコマンド（ターミナルから実行）

  claude                        Claude Codeを対話モードで起動
  claude "質問や指示"           一度だけ実行して終了（非対話モード）
  claude --help                 ヘルプを表示
  claude --version              バージョンを表示
  claude --model <モデルID>     使用するモデルを指定
  claude --print / -p           応答をプリントして終了（パイプ用途向け）
  claude --no-color             色なしで出力

----------------------------------------------------------------

■ スラッシュコマンド（チャット中に使用）

  /help                         ヘルプを表示
  /clear                        会話履歴をクリア
  /exit または /quit            Claude Codeを終了
  /model <モデルID>             使用するモデルを切り替え
  /fast                         Fast Mode（高速モード）をトグル
  /login                        Anthropicアカウントでログイン
  /logout                       ログアウト
  /status                       認証状態・モデルなどの現在の状態を表示
  /cost                         現在のセッションのトークン使用量とコストを表示
  /vim                          Vimキーバインドをトグル
  /terminal-setup               シェル統合をセットアップ
  /memory                       メモリファイル（CLAUDE.md）を編集
  /init                         プロジェクトにCLAUDE.mdを作成
  /review                       コードレビューを依頼
  /compact                      会話を圧縮して継続（コンテキストが長くなった場合）

----------------------------------------------------------------

■ キーボードショートカット

  Ctrl+C                        現在の処理を中断
  Ctrl+D                        終了
  Ctrl+L                        画面をクリア
  ↑ / ↓                        入力履歴を移動
  Tab                           コマンドやファイル名を補完

----------------------------------------------------------------

■ MCPサーバー管理

  claude mcp add <名前> <コマンド>    MCPサーバーを追加
  claude mcp list                     登録済みMCPサーバーを一覧表示
  claude mcp remove <名前>            MCPサーバーを削除

----------------------------------------------------------------

■ 設定ファイル

  CLAUDE.md                     プロジェクトのガイドライン（Claudeが自動的に読み込む）
  ~/.claude/settings.json       グローバル設定
  .claude/settings.json         プロジェクトレベルの設定

----------------------------------------------------------------

■ よく使う使い方の例

  # ファイルを渡して質問する
  cat error.log | claude "このエラーを解析して"

  # 特定ファイルを含めて実行
  claude --print "このコードを説明して" < main.py

  # 作業ディレクトリで起動（コードベース全体を参照できる）
  cd /path/to/project && claude
